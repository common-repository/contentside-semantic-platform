== Description ==

__Attention__ : cette extension fonctionne exclusivement avec une clé API fournie par [ContentSide](https://www.contentside.com/contact/)

== FONCTIONNALITÉS SEMANTIC PLATFORM ==
* Reconnaissance d’entités nommées

L’intelligence artificielle de l’extension Semantic Platform est capable d’identifier les entités nommées de vos articles. Une fois identifiées, elles sont contrôlées via votre dictionnaire.
Celles qui n’en font pas partie, vous sont suggérées, afin que vous puissiez enrichir votre dictionnaire.

* Recommandation de catégories

L’extension Semantic Platform est capable de rechercher, parmi vos catégories, celle(s) qui correspond(ent) le mieux à votre article.

* Suggestion d’articles similaires

_Sur l’ensemble de votre article_
L’extension Semantic Platform permet d’ajouter des liens vers des articles similaires suggérés par l’intelligence artificielle à partir de vos contenus.

_Sur une portion de votre article_
L’extension Semantic Platform vous permet de sélectionner un mot ou un ensemble de mots et de créer un lien vers un article similaire suggéré par l’intelligence artificielle..




== ÉDITER FACILEMENT ==
✔️ Automatiser les tâches fastidieuses
Parfaire la manière dont votre contenu sera visible en ligne nécessite un ensemble de tâches plutôt fastidieuses, mais pour autant essentielles. Grâce à l’intelligence artificielle, automatiser ces tâches vous permet un gain de temps et d’effort, décharge vos rédacteurs et vous assure un service de qualité experte.
✔️ Limiter le taux d’erreur
L’automatisation permet d’éviter un remplissage manuel erroné de données, et à l’inverse, de favoriser un résultat optimisé grâce à une vue d’ensemble de tous vos contenus.
✔️ Options de personnalisation
La personnalisation est un critère central pour se démarquer de vos concurrents. C’est pourquoi nous vous donnons la possibilité de choisir l’affichage des éléments que nous ajoutons à vos contenus pour les enrichir.

== RENFORCER LE RÉFÉRENCEMENT ==
✔️ Renforcer le maillage interne
Travailler le maillage de votre site web en insérant des liens pertinents entre vos articles similaires contribue fortement à l’amélioration de votre référencement.
✔️ Enrichir sémantiquement vos contenus
Pour que vos contenus soient bien référencés par les moteurs de recherche, ils doivent pouvoir identifier le sujet et les thématiques de vos articles.
✔️ Améliorer la visibilité de vos archives
Quelle qu’en soit la raison, vos contenus anciens tendent à être de moins en moins visibles sur votre site web, et par les moteurs de recherche. Pourtant, ils représentent un potentiel SEO immense pour votre site puisque votre longue traîne génère la majorité de votre trafic.

== AMÉLIORER L'EXPÉRIENCE UTILISATEUR ==
✔️ Simplifiez-leur la vie
Plus vos contenus et leur accès seront soignés, plus il sera facile pour un internaute de trouver l’information qu’il recherche sur votre site web
✔️ Obtenez leur satisfaction
Construire un site web dans lequel les internautes s’y retrouvent contribue largement à leur satisfaction. Proposez-leur une accessibilité optimale et fidélisez-les !

== ==

__Attention__ : l’extension Semantic Platform fonctionne si vous disposez déjà d’une taxonomie et de catégories pérennes. N’hésitez pas à nous consulter si vous avez besoin de conseils pour y parvenir au préalable.

== Installation ==

__Installation manuelle__ :

 * Télécharger l’extension Semantic Platform
 * Extraire l’archive ZIP
 * Téléverser le fichier sur le blog
 * Activer l’extension directement depuis WordPress
